About

Baby Watcher AI is a state of the art Technology that will help you understand the images that you have captured using the ultrasound scanner.

How to use?

  1. Click on start
    2. This will lead you to another page to upload the image

  2. Upload the image
    3. Please make sure you upload the images you took using the babywatcher ultrasound.

  3. Click Analyze
    4. This might take time as it is processed with the latest technologies. Sit tight as it gets analyzed.

  4. Enjoy your image
    5. The image you will receive will be the same image youi have selected but with bounding boxes that point to where the the found features are. These features can be head, spine and more if visible. The boxes will be labelled and will also has a confidence (How sure we are of the solution).
